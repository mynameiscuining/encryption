对称加密算法  
====
**基本介绍**  
对称加密算法是现在应用范围最广，使用频率最高的加密算法。  
对称的原理：   
加密密钥 = 解密密钥，加密运算是解密运算的逆运算。   
对称加密算法是初等的加密算法，从安全性上说，不是很高。  
常用的对称加密算法:DES(3DES)，AES，PBE，IDEA等。
